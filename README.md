Coders of Delhi

📌 Overview
This project analyzes a social network dataset to identify user relationships and recommend potential connections ("People You May Know").

## Tech Stack
* Python
* Pandas
* JSON

## Features
* Load and process user data
* Analyze friendships and connections
* Recommend new users based on mutual connections

## Files
* `Coders of Delhi.ipynb` → Main notebook
* `massive_data.json` → Dataset

## How to Run
1. Download the repository
2. Open the notebook in Jupyter Notebook or Google Colab
3. Run all cells

## Key Learning
* Data processing using JSON
* Basic recommendation system logic
* Social network analysis
